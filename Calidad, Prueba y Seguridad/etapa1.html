Servicio que ofrece QuickOrder

Elemento Descripción
Servicio Plataforma web para la gestión de pedidos de comida, que permite a los clientes registrarse, iniciar sesión, seleccionar platillos, pagar y recibir su pedido.
Consumidores
  • Clientes: Realizan pedidos y pagos en línea.
  • Cocineros: Reciben pedidos y preparan la comida.
  • Administrador: Agrega, edita y elimina platillos, y supervisa los pedidos.
  • Equipo TI: Mantiene el sistema operativo y seguro.
Valor que entrega
  • Agilidad: Elimina filas presenciales para ordenar.
  • Seguridad: Login con validación y recuperación de contraseña.
  • Eficiencia: La cocina recibe los pedidos en tiempo real.
  • Comodidad: Interfaz clara y oscura configurable según preferencia.

Identificación de Brechas ITIL

Aspecto Situación Actual
Gestión de incidentes Si ocurre una falla (login, pagos, caída de servidor), el equipo revisa manualmente y soluciona reiniciando el servicio o corrigiendo directamente en la base de datos.

No existe un canal formal de reporte para los usuarios.
Proceso de cambios Los cambios se realizan directamente sobre el código y la base de datos sin documentación formal.

No se registra quién solicitó el cambio ni se realizan pruebas antes de implementarlo en producción.
Medición del éxito No existen métricas automáticas.

El éxito se evalúa únicamente si el usuario logra completar su pedido y la cocina lo recibe.

No se registran estadísticas de tiempo de login, pedidos completados ni tasa de errores.
