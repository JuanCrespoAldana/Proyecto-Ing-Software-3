Gestión de Configuración

Se definen procesos para el control de versiones y configuración del proyecto.

  • Repositorio: GitHub como repositorio centralizado.
  • Versionado: Versionado semántico (v1.0.0, v1.1.0, v2.0.0).
  • Documentación: Wiki del repositorio con manuales y diagramas.
  • Backups: Copias semanales en la nube.
  • Estándares de código: Convenciones PSR para PHP.
  • Entorno: Diferenciación entre desarrollo, pruebas y producción.
