Escenarios de Casos de Uso

Crear Perfil

Nombre Crear Perfil
Descripción Permite registrar la información de los perfiles de usuario, incluyendo CódigoPersonal, Nombre, Apellido, Rol y Cédula.
Actor Administrador
Precondiciones El administrador debe estar autenticado.
Flujo Básico
ActorSistema
1. Ingresa al login2. El sistema muestra formulario de creación
3. Completa campos obligatorios4. El sistema valida la información
5. Clic en guardar6. Se almacena el perfil y muestra confirmación

Registrar Dispositivo

NombreRegistrar Dispositivo
DescripciónPermite asociar un dispositivo de monitoreo al usuario.
ActorUsuario
PrecondicionesEl usuario debe estar registrado y autenticado.
Flujo Básico
ActorSistema
1. El usuario accede a "Registrar Dispositivo"2. El sistema solicita ID del dispositivo
3. Ingresa el ID4. El sistema valida la existencia y lo asocia al usuario
5. Confirma registro6. Muestra mensaje de éxito

Generar Reporte

NombreGenerar Reporte
DescripciónEl sistema permite generar un reporte del estado de hidratación del usuario.
ActorUsuario
PrecondicionesEl dispositivo debe estar transmitiendo datos correctamente.
Flujo Básico
ActorSistema
1. Solicita generar reporte2. El sistema consulta los sensores
3. Espera confirmación4. Genera reporte en PDF/visual
5. Notifica al usuario y guarda el reporte

Recibir Notificación

NombreRecibir Notificación
DescripciónPermite al usuario o familiar recibir alertas sobre el nivel de hidratación.
ActorUsuario / Familiar
PrecondicionesDebe existir un reporte con nivel de riesgo crítico.
Flujo Básico
ActorSistema
1. Espera alertas2. El sistema evalúa parámetros
3. Envía notificación al móvil y al correo
4. Recibe notificación5. Registra envío en historial
